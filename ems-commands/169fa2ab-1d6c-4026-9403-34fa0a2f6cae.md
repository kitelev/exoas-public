# Create Action Instance

Command bound to `ems__ActionPrototype` layouts (binding `1d61b9cf`). Creates a
new `ems__Action` instance from the clicked prototype — the manual one-shot path
for the medication-intake prototypes (VL2: no auto-generation; the user presses
this when they take the supplement). Sibling of "Create Task Instance" (`bb00efed`).

- **Grounding**: `60d45933` (`create_instance`, targetClass `ems__Action`,
  prototype back-link via Universal IR `cf1ed702`).
- **Category**: creation. **Icon**: plus-square.
