# ems__Workflow

Workflow definition class. Each instance represents a finite-state-machine
describing how an `ems__Effort` (or subclass) progresses through statuses.

## Properties

- `ems__Workflow_targetClass` (1) — the asset class this workflow governs
- `ems__Workflow_initialState` (1) — starting status for new instances
- `ems__Workflow_terminalStates` (0..N) — end statuses where workflow halts
- `ems__Workflow_isDefault` (0..1, boolean) — class-default flag

## Discovery

`WorkflowResolver.findDefaultWorkflow(assetClass)` walks
`?w rdf:type ems:Workflow ; ems:Workflow_targetClass ?c ; ems:Workflow_isDefault true`.
First match wins.
