# ems__WorkflowStateCategory

Catalog of workflow state categories. Named-individual enum with three
fixed members:

- `ems__WorkflowStateCategoryInitial` — starting state (e.g. Draft)
- `ems__WorkflowStateCategoryActive` — in-progress (Backlog, Doing, etc.)
- `ems__WorkflowStateCategoryTerminal` — end (Done, Trashed)

## Pattern

Follows the same catalog homoiconization pattern as `ems__EffortStatus`
(UID-identity dispatch in TS) and `exocmd__GroundingType`. Categories
exposed via `ems__WorkflowState_category`.
