# ems__WorkflowState

A node in a Workflow's state machine. Binds a status value to its workflow
context plus presentation/category metadata.

## Properties

- `ems__WorkflowState_workflow` (1) — parent Workflow
- `ems__WorkflowState_status` (1) — `ems__EffortStatus` value
- `ems__WorkflowState_order` (1, integer) — display position
- `ems__WorkflowState_optional` (0..1, boolean) — can be skipped
- `ems__WorkflowState_category` (1) — `ems__WorkflowStateCategory` named individual (Initial / Active / Terminal). RFC 36347daf Phase 2.
- `ems__WorkflowState_timestampOnEnter` (0..N) — legacy property-name hints (kept for backward compat with `EffortStatusWorkflow` facade; Phase 2 prefers per-transition `postActions`)
- `ems__WorkflowState_badgeColor` (0..1) — UI hint
