# ems__ActionPrototype

Prototype-template class for atomic `ems__Action` efforts (epic `40188c71`).

Sibling of `ems__TaskPrototype` (`df7e579d`) — same three-parent shape:
`ems__EffortPrototype` (`a5c6d3aa`) + the instance class `ems__Action` (`6a99d2ca`) + `exo__Prototype` (`ebf717aa`).

Unlike `ems__TaskPrototype`, an `ems__ActionPrototype` is **not** combined with
`ems__Recurring` (VL2 2026-06-28): Action prototypes are manual one-shot templates
— no auto-generation of daily instances. The user creates an `ems__Action`
instance from the prototype on demand (the "Create Action Instance" command bound
to `ems__ActionPrototype` layouts).
