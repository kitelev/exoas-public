# Projects without active children

Dogfood-запрос для RFC c78cc5c8 Phase 1a. Возвращает ассеты класса `ems__Project`, у которых нет ни одного child-эффорта в активных статусах (Backlog / ToDo / Doing). Проекты с zero-children тоже включаются.

```sparql
PREFIX exo: <https://exocortex.my/ontology/exo#>
PREFIX ems: <https://exocortex.my/ontology/ems#>

SELECT ?project ?label
WHERE {
  ?project exo:Instance_class <https://exocortex.my/asset/7db5eeff-718a-49b0-8d2b-39b084a356e3> .
  OPTIONAL { ?project exo:Asset_label ?label . }
  FILTER NOT EXISTS {
    ?child ems:Effort_parent ?project .
    ?child ems:Effort_status ?status .
    FILTER (?status IN (
      <https://exocortex.my/asset/753a44d5-846c-4b82-9196-4fd9a4d48777>,
      <https://exocortex.my/asset/6a0e933a-6653-46f4-95ae-ed7508177c73>,
      <https://exocortex.my/asset/027e78f4-6e16-4b36-b8fb-5510507d5745>
    ))
  }
}
ORDER BY ?label
```
