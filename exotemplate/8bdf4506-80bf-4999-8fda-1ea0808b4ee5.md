# exotemplate__Template

Авторь шаблон как обычную заметку: во frontmatter укажи `exo__Instance_class` = этот класс, а **тело** заметки — содержимое шаблона. Можно использовать `$token`-маркеры (`$today`, `$nowTimestamp`, `$randomUUIDv4`, …) — они резолвятся при вставке через общий `SubstitutionResolverRegistry` (те же `exocmd__SubstitutionToken`, Веха 4).

## Пример

```markdown
---
exo__Instance_class: "[[8bdf4506-80bf-4999-8fda-1ea0808b4ee5]]"
exo__Asset_label: Project body
---
## Полезные ресурсы

## План выполнения

## Log
- $nowTimestamp — created
```
