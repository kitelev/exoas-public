# Plan on today (declarative)

Sets `ems__Effort_plannedStartTimestamp` on the target asset to today at local
midnight (`YYYY-MM-DDT00:00:00`) via the `$todayStart` substitution token
resolved by `GroundingExecutor.substituteVariables`.

Issue #3136 (RFC `18407cb2-9554-4897-9213-17321f9dd434` Path B, Q3.b closure)
— migrated from `service_call` + `serviceId: planOnToday` to declarative
`property_set` once the `$todayStart` token landed in the engine. Byte-for-byte
identical output to the previous `TaskStatusService.planOnToday` service call.
