# Shift day backward via property_shift

Declarative grounding that shifts `ems__Effort_plannedStartTimestamp` backward
by one day (`-P1D`). Replaces the previous `service_call` / `shiftDay`
(direction=backward) form (Issue #3134 — Homoiconicity Invariant Q1 closure,
RFC `18407cb2-9554-4897-9213-17321f9dd434` Path B).
