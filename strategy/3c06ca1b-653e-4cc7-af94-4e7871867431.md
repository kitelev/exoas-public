## Описание

- strategy__Initiative — workstream под strategy__Pillar (или Foundation Layer transversal)
	- Atomic unit of strategic work, decomposable в ems__Project и далее ems__Task per PMBOK
- Properties:
	- strategy__Initiative_pillar — wikilink на parent Pillar (optional для Foundation initiatives)
	- strategy__Initiative_layer — string enum: foundation-critical / foundation-leverage / pillar
	- strategy__Initiative_id — string label (например I1.1, F2)
	- strategy__Initiative_horizon — string sprint/quarter (S0, S1-2, Q4, 2027-Q1)
	- strategy__Initiative_outcome — string description initiative-level outcome
	- strategy__Initiative_status — string status (planned, in-progress, done, blocked)
